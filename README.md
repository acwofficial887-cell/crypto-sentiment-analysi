# Crypto Market Sentiment Analysis

## Project Overview
This project analyzes the Fear & Greed Index dataset (2018–2025) to understand crypto market sentiment.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL (SQLite in Colab)
- Looker Studio

## Key Insights
- Fear sentiment is higher than greed sentiment
- Extreme market conditions indicate strong volatility
- Average sentiment remains close to neutral over the long term

## Files in this Repository
- Dashboard PDF
- Dataset file
- Project documentation
